### Hi there 👋


      2021년까지의 작업 내역은 https://joilnam.github.io/portfollio/ 에 기록했습니다.

      maria db, mysql , mongo db, firebase에 db server를 설계 해보며
      백엔드 구조를 익히기 위해 지속적으로 학습했습니다.

      이후엔 React와 Vue Js, PHP로 구성된 웹페이지 제작연습을 개인적으로 진행 해왔습니다.
